Асинхронний код, задача

Правда чи ні, що при читанні файла за допомогою readFileSync можуть виникнути наступні проблеми:

Варіант1:
Під час читання файлу не обробляються системні таймери.
Відповідь: так: Основний потік виконання коду блокується до тих пір, доки файл не буде прочитаний.

Варіант2:
Виконання коду переривається, якщо файл по вказаному шляху не знайдений.
Відповідь: так: У випадку асинхронного читання, код виконається до кінця, помилка про відсутність файлу буде передана в функцію зворотнього виклику (callback).

Варіант3:
Під час читання файлу не обробляються запити від користувача у випадку, коли додаток є сервісом.
Відповідь: так: Основний потік виконання коду блокується до тих пір, доки файл не буде прочитаний.

Варіант4:
Одночасне читання декількох файлів працює повільніше.
Відповідь: ні
