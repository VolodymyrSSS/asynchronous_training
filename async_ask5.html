Асинхронный код, задача

При чтении файла с помощью readFileSync возникают следующие проблемы:

Вариант1:
Во время чтения файла не обрабатываются системные таймеры
Ответ: yes:Основной поток исполнения кода блокируется до тех пор, пока файл не будет прочитан

Вариант2:
Выполнение кода прерывается, если файл по указанному пути не найден
Ответ: yes:В случае асинхронного чтения код выполнится до конца, ошибка об отсутствии файла будет передана в функцию обратного вызова (callback).

Вариант3:
Во время чтения файла не обрабатываются запросы от пользователя в случае, когда приложение является сервером
Ответ: yes:Основной поток исполнения кода блокируется до тех пор, пока файл не будет прочитан

Вариант4:
Одновременное чтение нескольких файлов работает медленнее
Ответ: no
