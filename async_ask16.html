Асинхронный код, задача

Встроенная функция setTimeout использует колбэк-функции. Создайте альтернативу, использующую промисы. Работайте с кодом ниже.


function delay(ms) {
// ваш код
}

delay(3000).then(() => console.log('выполнилось через 3 секунды'));


Решение:


function delay(ms) {
return new Promise(resolve => setTimeout(resolve, ms));
}

delay(3000).then(() => console.log('выполнилось через 3 секунды'));


Заметьте, что resolve вызывается без аргументов. Мы не возвращаем из delay ничего, просто гарантируем задержку.
