Асинхронний код, задача

Вбудована функція setTimeout використовує колбек-функцію.
Створіть альтернативу, яка використовує проміс.



function delay(ms) {
   // ваш код
}

delay(3000).then(() => console.log('виконалось через 3 секунди'));


Рішення:


function delay(ms) {
   return new Promise(resolve => setTimeout(resolve, ms));
}

delay(3000).then(() => console.log('виконалось через 3 секунди'));


Зауважте, що resolve викликається без аргументів. Ми не повертаєм з delay нічого, просто гарантуємо затримку.
