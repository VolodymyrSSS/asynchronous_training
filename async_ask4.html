Асинхронний код, задача

Скільки разів буде викликана функція func якщо виконати
setInterval(func, 5000)?

function toStove() {
   return 'Покласти на плиту';
}
function fromStove() {
   return 'Зняти з плити';
}

function toStir() {
   return 'Помішувати';
}

toStove();
setInterval(toStir, 5000);

Варіант1:
Жодного разу, тому що після виклику setInterval програма закінчиться.
Відповідь: ні: До тих пір, доки інтерпритатор знає про системний таймер, виконання коду не завершиться. Функція func буде викликатись кожні 5 секунд.

Варіант2:
1 раз після того як пройде п'ять секунд.
Відповідь: ні: Функція func буде викликатись кажні 5 секунд, до тих пір, доки системний таймер не зупинять.

Варіант3:
Необмежену кількість разів, через кожні п'ять секунд.
Відповідь: так: Функція func буде викликатись до тих пір, доки системний таймер не зупинять.

Варіант4:
5000 разів.
Відповідь: ні: В об'явленні setInterval(func, 5000) другим параметром вказується інтервал, через який потрібно викликати функцію колбека. А отже функція func буде викликатись кожні 5 секунд до тих пір, доки системний таймер не зупинять.
