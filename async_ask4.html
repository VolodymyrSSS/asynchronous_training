Сколько раз будет вызвана функция func если выполнить
setInterval(func, 1000)?

function toStove() {
return 'Поставить на плиту';
}
function fromStove() {
return 'Снять с плиты';
}

function toStir() {
return 'Помешивать';
}

toStove();
setInterval(toStir, 1000);

Вариант1:
ни разу, потому что после вызова setInterval программа завершится.
Ответ: no: До тех пор, пока интерпретатор знает о системном таймере, выполнение кода не завершится. Функция func будет вызываться каждые 5 секунд.

Вариант2:
1 раз по прошествии пяти секунд.
Ответ: no:Функция func будет вызываться каждые 5 секунд, до тех пор, пока системный таймер не остановят.

Вариант3:
Неограниченное число раз, каждые пять секунд.
Ответ: yes:Функция func будет вызываться до тех пор, пока системный таймер не остановят.

Вариант4:
5000 раз.
Ответ: no:В объявлении setInterval(func, 5000) вторым параметром указывается интервал, через который нужно вызывать колбэк. Таким образом, функция func будет вызываться каждые 5 секунд, до тех пор, пока системный таймер не остановят.
