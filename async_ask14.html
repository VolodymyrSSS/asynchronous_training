Асинхронний код, задача

Чи можна повторно виконати проміс? Що буде виведено ?



let promise = new Promise(function(resolve, reject) {
   resolve(1);

   setTimeout(() => resolve(2), 1000);
});

promise.then(console.log); // ???


Відповідь: В консолі буде виведено 1.


Ми маємо два виклики resolve. Але першим буде виведений resolve(1).
Інший виклик resolve буде проігнорований, оскільки враховується тільки перший виклик reject/resolve. Усі наступні виклики ігноруються.
