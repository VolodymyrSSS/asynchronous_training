Асинхронний код, самоперевірка

Що буде в стеку викликів у момент виклику second()?

function first() {
   console.log('first');
}

function second() {
   console.log('second');
}

first();
second(); // Що буде в стеку викликів у цей момент?

Варіант1:
anonymous та first
Відповідь: ні: Функція anonymous опинилась в стеку коли код почав виконуватися. Функція first не буде в стеку викликів, тому що її виконання вже завершено.

Варіант2:
anonymous і second
Відповідь: так: Функція anonymous опинилась в стеку коли код почав виконуватися. second лежить в стеку виконання, тому що вона виконується в цей момент.

Вариант3:
first та second
Відповідь: ні: Функція first не буде в стеку викликів, тому що її вкиконання вже завершиться на той час.

Вариант4:
anonymous, first та second
Відповідь: ні: Функція first не буде в стеку викликів, тому що її вкиконання вже завершено.
