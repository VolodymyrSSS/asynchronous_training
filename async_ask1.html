Асинхронный код, задача

Что будет в стеке вызовов, в момент вызова second()?

function first() {
console.log('first');
}

function second() {
console.log('second');
}

first();
second(); // Что будет в стеке вызовов в этот момент?

Вариант1:
anonymous и first
Ответ: no: Функция anonymous оказалась в стеке когда код начал исполняться. Функции first не будет в стеке вызовов, потому что её выполнение уже завершено.

Вариант2:
anonymous и second
Ответ: yes: Функция anonymous оказалась в стеке когда код начал исполняться. second лежит в стеке вызовов, потому что выполняется в этот момент.

Вариант3:
first и second
Ответ: no: Функции first не будет в стеке вызовов, потому что её выполнение уже завершено.

Вариант4:
anonymous, first и second
Ответ: no: Функции first не будет в стеке вызовов, потому что её выполнение уже завершено.
