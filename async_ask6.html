Асинхронний код, задача

Функція 'sayHi' виводить вираз 'Привіт' у консоль.
Потрібно вивести цей вираз через 1,5 секунди після запуску цієї функції.

function sayHi() {
   console.log('Привіт');
}


Рішення: Для цього використовуємо метод 'setTimeout' де першим аргументом вкажемо назву функції, а другим - потрібну затримку в мілісекундах. При цьому слід пам'ятати, що якщо додати скобки після функції (тобто її викликати), то це не спрацює, бо setTimeout очікує посилання на функцію, а не її результат. Саме sayHi() запускає виконання функції, і результат виконання відправляється в setTimeout. У нашому випадку результатом виконання sayHi() є undefined (бо функція нічого не повертає), а тому нічого не планується. А в консолі виведеться через 1.5 секунди вираз "Привіт".


function sayHi() {
console.log('Привіт');
}

setTimeout(sayHi, 1500); // Привіт


Зауважте, якщо перший аргумент є рядок то JavaScript створює з нього функцію.
А якщо передати функцію замість рядка (лапки при цьому зберігаються), це також буде працювати (тут зробили затримку в 2 секунди):


setTimeout( "console.log('Hello, hello!!!')", 2000 ); // Hello, hello!!! іn 2 sec


Загалом, використання рядків не рекомендують використовувати в setTimeout. Тут замість рядків краще використовуйте функцію як вказано в попередньому прикладі. А ось ще один приклад з функцією колбеку при цьому лапок не потрібно в такій функції:


setTimeout( () => console.log('Вітання!'), 2500 ); // Вітання! через 2.5 сек
