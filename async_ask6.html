Асинхронный код, задача

Функция 'sayHi' выводит фразу 'Привет' в консоль.
Надо вывести эту фразу через 1,5 секунды после запуска этой функции

function sayHi() {
console.log('Привет');
}


Ответ: Для этого используем метод 'setTimeout' где первым аргументом укажем название функции,а вторым - нужную задержку в милисекундах. При этом следует заметить, что если добавить скобки после функции, то это не работает, потому что setTimeout ожидает ссылку на функцию. Здесь sayHi() запускает выполнение функции, и результат выполнения отправляется в setTimeout. В нашем случае результатом выполнения sayHi() является undefined (так как функция ничего не возвращает), поэтому ничего не планируется.


function sayHi() {
console.log('Привет');
}

setTimeout(sayHi, 1500); // Привет


При этом, если первый аргумент является строкой, то JavaScript создаст из неё функцию.
Это также будет работать (здесь сделали задержку 2 секунды):


function sayHi() {
console.log('Привет');
}

setTimeout("console.log('Привет')", 2000); // Привет


Но использование строк не рекомендуется. Вместо этого используйте функции. Например, так:


function sayHi() {
console.log('Привет');
}

setTimeout(() => console.log('Привет'), 2500); // Привет
