Асинхронний код, задача

Використовуючи функцію 'sayHi' та аргументи, які передаються разом з цією функцією (вираз вітання та ім'я) виведіть в консоль слово-вираз 'Привіт Модест' через 1 секунду після її запуску.

function sayHi(phrase, who) {
   console.log( phrase + ', ' + who + " !");
}


Рішення: Для цього використаємо метод setTimeout де першим аргументом вкажемо назву функції (без скобок бо це посилання); другим - потрібну затримку в мілісекундах; третім аргументом - рядок (перший вираз для цілого слово-виразу), четвертим аргументом - рядок (другий вираз - ім'я для цілого слово-виразу). Рішення має вид:


function sayHi(phrase, who) {
console.log( phrase + ', ' + who + " !");
}

setTimeout(sayHi, 1000, "Привіт", "Модест"); // Привіт, Модест через 1 сек
