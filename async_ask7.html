Асинхронный код, задача

Используя функцию 'sayHi' и аргументы передаваемые с этой функцией (фразу и имя) вывести в консоли выражение 'Привет Модест' через 1 секунду после её запуска.

function sayHi(phrase, who) {
console.log( phrase + ', ' + who + " !");
}


Ответ: Для этого используем метод 'setTimeout' где первым аргументом укажем название функции без скобок; вторым - нужную задержку в милисекундах; третим аргументом - нужную фразу, ну и четвёртым аргументом - имя.


function sayHi(phrase, who) {
console.log( phrase + ', ' + who + " !");
}

setTimeout(sayHi, 1000, "Привет", "Модест"); // Привет, Модест
