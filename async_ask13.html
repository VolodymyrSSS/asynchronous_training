Асинхронний код, задача

В даному коді є запланований виклик setTimeout, а потім виконується складне обчислення, для завершення якого потрібно більше ніж 100 мс.
Коли буде виконана запланована функція?
1. Перед циклом; 2. На початку цикла; 3. Після циклу.
Що покаже console.log ?


let i = 0;

setTimeout(() => console.log(i), 100); // ?

// уявімо, що час виконання цієї функції > 100 мс
for(let j = 0; j < 100000000; j++) {
   i++;
}


Варіант1:
Перед циклом.
Відповідь: ні: Будь-який виклик setTimeout буде виконаний після того, як поточний код завершиться.

Варіант2:
На початку циклу.
Відповідь: ні: Будь-який виклик setTimeout буде виконаний після того, як поточний код завершиться.

Варіант3:
Після цикла.
Відповідь: так: Будь-який виклик setTimeout буде виконаний після того, як поточний код завершиться.


Останнім значенням "i" буде 100000000.
