Асинхронный код, задача

Что будет в браузере идентификатором таймера ?
А примит идентификатором таймера значение null после его отмены ?


let timerId = setTimeout(() => console.log("ничего не происходит"), 1500);

clearTimeout(timerId);


Ответ: Для этого, зададим идентификатор таймера в переменную и выведем её в консоли.
Как мы видим, в браузере идентификатором таймера является число.
Потом, отменим таймер с помощью метода clearTimeout() задав аргументом


let timerId = setTimeout(() => console.log("ничего не происходит"), 1500);
console.log(timerId); // идентификатор таймера - у нас это число 1

clearTimeout(timerId);
console.log(timerId); // число 1 - тот же идентификатор (не принимает значение null после отмены);


В других средах идентификатором таймера может быть что-то ещё. Например, Node.js возвращает объект таймера с дополнительными методами.
