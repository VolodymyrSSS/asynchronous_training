Асинхронний код, задача

Що буде виведено у браузері від ідентифікатора таймера ?
А чи стане інентифікатор таймера значенням null після його відміни ?


Відповідь: Для цього, давайте присвоємо ідентифікатор таймера якійсь змінній і потім виведемо значення цьої змінної в консоль.
В результаті, в браузері ідентифікатор таймера - це число "1".
Потім, відмінимо таймер методом clearTimeout() куда аргументом передамо той самий ідентифікатор таймера. В результаті - значення ідентифікатора таймераа не змінилось, а залишилось "1".


let timerId = setTimeout(() => console.log("нічого не діється"), 1500);
console.log(timerId); // це число 1 - ідентифікатор таймера

clearTimeout(timerId); // аргументом передано той самий ідентифікатор
console.log(timerId); // число 1 - не стало значення null після відміни
