Асинхронный код, задача

Функция 'tickRun' выводит фразу 'tick' в консоль.
Вывести сообщение 'tick' каждые 1.5 секунды. Через 5 секунд вывод прекратить и вывести фразу 'stop'.


function tickRun() {
console.log('tick');
}



Ответ: Для этого используем метод 'setInterval' который запускает функцию не один раз, а периодически через указанный интервал времени. В этом методе первым аргументом укажем название функции, а вторым - нужную задержку в милисекундах.
Но если задать просто метод 'setInterval' вот так:


setInterval(tickWork, 1500);
то 'tick' будет выводиться бескончно каждые 1.5 секунды пока не остановят


Поэтому, необходимо остановить дальнейшее выполнение функции, а для этого надо вызвать метод clearInterval(timerId), где timerId - это идентификатор таймера в виде переменной.



повторять с интервалом 1.5 секунды
let timerId = setInterval(() => console.log("tick"), 1500);


Ну и дальше, определим в методе 'setTimeout' временной интервал в 5 секунд где первым аргументом укажем метод остановки 'clearInterval', а вторым - нужную задержку для его выполнения.



остановить вывод через 5 секунд
setTimeout(() => { clearInterval(timerId); console.log("stop"); }, 5000);
